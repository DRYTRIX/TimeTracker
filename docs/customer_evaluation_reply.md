# Customer evaluation — draft reply

**Subject:** Re: TimeTracker evaluation — your questions answered

Hello,

Thank you for the follow-up and for continuing to evaluate TimeTracker. Below are answers to each of your points.

---

## 1. Notification when workday was not started + corrections

**Notification:** Enable a **missed workday reminder** under **Settings → Notifications**:

- **In-app / browser push:** enable *Smart notifications*, then *Remind me if I forgot to start my workday* (default reminder time 09:30 in your timezone).
- **Email:** enable *Email me if I forgot to start my workday*.

The reminder only fires on expected working days (Monday–Friday, excluding company holidays and approved time off) when **Start Workday** has not been used.

**Correction:** If someone forgot to clock in:

1. **Same day:** press **Start Workday**, then open **Attendance history** and request a correction to adjust the start time.
2. **Past day:** use **Request missing workday** on the attendance history page. An admin reviews under **Admin → Attendance corrections**.

---

## 2. PDF for time-off / vacation requests

Each time-off request on the Workforce dashboard has a **PDF** button. The form includes employee details, dates, comments, and signature lines. When approved, the approver name and date are printed.

**Note:** This is a printable form with approval metadata, not a cryptographic e-signature.

---

## 3. Android app — Start Workday button

This was not intentional — older APKs lacked the feature. The current mobile app includes **Start workday** on the **Home** and **Timer** screens. Please install the latest build.

---

## 4. Holidays and time-off on the calendar

The **Calendar** view shows **company holidays** and your **time-off** as coloured day bands. Use the **Holidays** and **Time Off** filter checkboxes. Holidays are managed under **Workforce** (admin).

---

## 5. Timesheet periods

Periods are more than informational: **submit → approve/reject → close**. **Closing** locks time entries and attendance days. Payroll CSV export can filter to approved/closed weeks. No automatic emails are sent today.
